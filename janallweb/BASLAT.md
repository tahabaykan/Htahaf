## ✅ Backend ÇALIŞIYOR!

Backend başarıyla çalışıyor: `http://127.0.0.1:5000`

## 📋 Şimdi Yapılacaklar

### 1. Backend Terminal'ini Açık Tut
Backend çalışan terminal'i **AÇIK BIRAK** (Ctrl+C ile durdurma!)

### 2. Yeni Terminal Aç ve Frontend'i Başlat

**Yeni bir PowerShell/CMD terminal aç** ve şunu çalıştır:

```bash
cd C:\Users\User\OneDrive\Masaüstü\Proje\StockTracker\janallweb\frontend
npm install
npm run dev
```

### 3. Tarayıcıda Aç

Frontend başladıktan sonra tarayıcıda şu adrese git:
```
http://127.0.0.1:3000
```

## 🎯 Kullanım

1. **Hammer Pro'ya Bağlan**: Sağ üstteki "Bağlan" butonuna tıkla
2. **CSV Yükle**: Ana sayfada CSV dosyası yükle
3. **Pozisyonları Gör**: Pozisyonlar sayfasına git
4. **Emir Gönder**: Emirler sayfasından yeni emir gönder
